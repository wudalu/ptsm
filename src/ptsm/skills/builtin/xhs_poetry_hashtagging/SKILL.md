# XHS Poetry Hashtagging

在小红书诗词赏析文案中：

1. 标签必须包含 `#苏轼`。
2. 另外补 2 到 3 个阅读、心境或场景标签。
3. 标签要自然，不要写成泛泛的国学关键词堆砌。
